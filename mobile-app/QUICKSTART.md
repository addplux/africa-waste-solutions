# Quick Start Guide

## 1. Install Dependencies

```bash
cd mobile-app
npx expo install
```

## 2. Update API URL

Edit `constants/Config.js`:
```javascript
BASE_URL: 'https://YOUR-RAILWAY-APP.railway.app/api'
```

## 3. Run the App

```bash
npm start
```

Then press `a` for Android or scan QR code with Expo Go app.

## 4. Test Login

Use your existing backend credentials to login!
